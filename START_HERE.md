## What is this?

A static website about Easy Language (Liicht Sprooch) in Luxembourg.

- **URL:** https://liichtsprooch.lu
- **Hosting:** GitHub Pages (free)
- **Auto-deployment:** GitHub Actions builds on every push

## How It Works

```
Edit markdown files → Commit & Push → GitHub Actions builds → Live in 2 minutes
```

Everything happens automatically. You just edit and push.

---

## Key Files

| File | Purpose |
|------|---------|
| `assets/artikelen/` | Markdown articles (edit these!) |
| `config.php` | Article metadata & navigation |
| `docs/` | Generated static site (auto-created) |
| `build.php` | Static site generator (runs automatically) |
